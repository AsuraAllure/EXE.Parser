Задание: реализация аналога dumpbin.

Требования:
CLI.
Выдача заголовков PE-файла.
Выдача секций.
Таблицы импорта / экспорта.
Набор текстовых бинарников.

Формат использования: в командной строке выполнить:
python main.py example.exe
В командную строку напечатается дамп файла.
